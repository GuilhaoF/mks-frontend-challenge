### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/GuilhaoF/mks-frontend-challenge
   ```
2. Install NPM packages
   ```sh
   npm install
   or
   yarn / yarn add
   ```

### Run

```sh
npm run dev
or
yarn dev
```
